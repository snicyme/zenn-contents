## chiikobot - ちいこぼっと

Twitch の簡易チャットボットです。
「ちいこ」とは「ちいこい」から来てます、「小さい」ということです。小さなボットなので「TinyBot」等がいいかな？と思ったのですが [Twitch](https://twitch.tv) に同名のアカウントがありましたのでアレンジしてみました。

Glitch にて動作させていますが、Node.js が走る環境であれば大抵動くと思います。

:::message
公式 Bot は私が登録した方のみ使用可能なので、興味のある方は「配信している時」にでも声をかけてください。以下で配信しています

## 出来ること - 配信者用

### カウントを表示

:::details 表示

```PlainText
!count {{userName}}
```

**オプション**

- `{{userName}}`

  - 表示する配信者さんの Twitch ID

**使用例**

```PlainText
!count snicyme
```

:::

### カウントリセット

:::details ユーザーを指定してリセット

```PlainText
!cr {{userName}} {{count?}}
```

**オプション**

- `{{userName}}`

  - リセット対象のユーザー ID

- `{{count?}}`

  - 設定するカウント数
  - 文字通りリセットなら 0
  - 指定しなければ自動で 0

**使用例**

```PlainText
!cr snicyme <- 0 になる
!cr snicyme 0 <- これも 0
!cr snicyme 49 <- これは 49
```

:::

:::details 全部リセット

全員 0 スタートだ！なお、オプションはない模様

```PlainText
!crall
```

:::

### 視聴者のコメント数に応じてメッセージ表示

:::details 一覧

既に設定されているメッセージ一覧が表示されます

```PlainText
!cm
```

:::

:::details 追加

```PlainText
!cm {{threshold}} {{name}} {{message}}
```

**オプション**

- `{{threshold}}`

  - メッセージを表示するタイミング
  - 視聴者さんのコメント数がこの値に到達した時にメッセージが表示されます

- `{{name}}`

  - 設定名
  - !cm 単体で呼び出した際にわかりやすくするための設定名です

- `{{message}}`

  - 表示するメッセージ
  - 本文には以下の値が使用可能
    - `{{userName}}` -> Twitch ID
    - `{{displayName}}` -> 表示名

**使用例**

```PlainText
!cm 1 初コメ {{displayName}} さん！初めてのコメントありがとうございます😄 {{displayName}} さんの素敵なチャンネルはこちら -> twitch.tv/{{userName}}
```

**注意点**

既に threshold = 1 の設定がある場合、threshold = 1 を指定すると既存の設定を上書きしてしまいます

:::

:::details 削除

```PlainText
!rmcm {{threshold}}
```

**オプション**

- `{{threshold}}`
  - メッセージを表示するタイミング
  - 設定で指定した値

**使用例**

```PlainText
!rmcm 1
```

:::

### 無視無視メッセージ

:::details 一覧

一覧では自動で付加された ID が表示されます。削除の時に必要です。

```PlainText
!filter
```

:::

:::details 追加

```PlainText
!filter {{pattern}}
```

**オプション**

- `{{pattern}}`

  - この内容が含まれている場合無視します
  - 一応内部で new RegExp({{pattern}})としていますので正規表現が一部通るかな？

**使用例**

```PlainText
!filter viewers
!filter followers
```

:::

:::details 削除

```PlainText
!rmfilter {{id}}
```

**オプション**

- `{{id}}`

  - 設定の ID。一覧表示で確認できます、長いです 😇

**使用例**

```PlainText
!rmfilter 51a19a5a-f058-4e78-b0db-d28d131731a8
```

:::

### ボットの有効化/無効化

:::details 無効化

!resume 以外のコマンドを受け付けなくなり、カウントもしなくなります。

```PlainText
!suspend
```

:::

:::details 有効化

```PlainText
!resume
```

:::

### ボットに配信者を登録/接続/切断

:::details 登録

登録すると自動で接続されます

```PlainText
!regch {{userName}}
```

**オプション**

- `{{userName}}`

  - 登録する配信者さんの Twitch ID

:::

:::details 接続

```PlainText
!join {{userName}}
```

**オプション**

- `{{userName}}`

  - 接続する配信者さんの Twitch ID

:::

:::details 切断

```PlainText
!leave {{userName}}
```

**オプション**

- `{{userName}}`

  - 切断する配信者さんの Twitch ID

:::
